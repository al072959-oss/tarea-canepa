Acordeón de Etiquetas HTML

Define la cabecera de un documento o una sección. Es comúnmente usada para contener logotipos, menús de navegación, títulos, entre otros elementos.

Ejemplo:

<header>
    <h1>Bienvenido a mi página</h1>
    <nav>...

Define una sección de navegación. Se utiliza para enlazar a otras páginas o secciones dentro de un mismo documento.

Ejemplo:

<nav>
    <a href="#inicio">Inicio</a>
    <a href="#contacto">Contacto</a>
</nav>

Representa el contenido principal de un documento. El contenido que es único y central para la página web debe ir dentro de esta etiqueta.

Ejemplo:

<main>
    <h2>Artículo Principal</h2>
    <p>Contenido principal de la página</p>
</main>

Define contenido relacionado que puede ser considerado adicional o complementario al contenido principal, como barras laterales o citas.

Ejemplo:

<aside>
    <p>Este es un contenido relacionado</p>
</aside>

Representa el pie de página de un documento o sección. Se suele usar para poner información de contacto, derechos de autor o enlaces relacionados.

Ejemplo:

<footer>
    <p>© 2025 Mi Página Web</p>
</footer>

Definen encabezados de diferentes niveles, siendo `

` el más importante y `

` el menos importante.

Ejemplo:

<h1>Encabezado principal</h1>
<h2>Subencabezado</h2>

La etiqueta `` es un contenedor en línea que se utiliza para agrupar contenido en línea y poder darle estilo.

Ejemplo:

<span style="color:red">Texto en rojo</span>

La etiqueta `

` es un contenedor genérico utilizado para agrupar contenido en bloques.

Ejemplo:

<div class="contenedor">
    <p>Este es un bloque</p>
</div>

Define un hipervínculo, utilizado para enlazar a otras páginas o secciones.

Ejemplo:

<a href="https://www.ejemplo.com">Ir a ejemplo</a>

Se utilizan para crear listas ordenadas (`

    `) o desordenadas (`
      `). Los elementos de la lista se definen con `
    • `.

      Ejemplo:

      <ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
